# grind-tracker

A daily coding progress tracker for developers who grind after their day job.

> "Built by someone who codes after a 14-hour shift. This app tracks that grind."

## Features
- User authentication (register, login, logout)
- Log daily entries: hours coded, applications sent, notes  
- Dashboard: streak tracking, total stats, Chart.js visualisation
- Edit and delete entries
- Profile page
- Full REST API with Django REST Framework

## Stack
Python · Django · DRF · SQLite · HTML · CSS · JavaScript · Bootstrap 5 · Chart.js

## Run locally
```bash
git clone https://github.com/NkosinathiMkhonza/grind-tracker.git
cd grind-tracker
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
